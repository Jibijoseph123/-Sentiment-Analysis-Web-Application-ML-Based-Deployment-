# Sentiment Analysis Web Application (ML-Based)

## Overview
This project is a sentiment analysis system built using Machine Learning techniques. It classifies movie reviews as positive or negative using TF-IDF and Logistic Regression.

## Dataset
IMDB 50K Movie Reviews Dataset from Kaggle.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Streamlit
- NLP (TF-IDF)

## Features
- Text preprocessing
- TF-IDF vectorization
- Logistic Regression classification
- Web deployment using Streamlit

## How to Run
1. Install dependencies:
   pip install streamlit scikit-learn pandas numpy

2. Run the app:
   py -m streamlit run app.py
